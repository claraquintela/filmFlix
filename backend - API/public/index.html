API Films

Documentation

Cette API permet d'afficher, d'ajouter, de modifier e de supprimer des films de la base de données.

Lien site : https://four1f-tp1-claraquintela-1.onrender.com

Lien Github

Routes

Films

* Récupérer la liste de tous les films
GET api/films
https://four1f-tp1-claraquintela-1.onrender.com/api/films
* Ajouter un nouveau film
POST api/films
https://four1f-tp1-claraquintela-1.onrender.com/api/films
Pametres requis: un objet film dans le corps de la rêquete: 
              let film = {
                titre: "titre",
                description: "description",
                annee: 2024,
                realisation: "nom du réalisateur",
                genre: "genre du film",
                titreVignette: "titre de la vignette"
              }
* Récupérer un film spécifique
GET api/films/{id}
https://four1f-tp1-claraquintela-1.onrender.com/api/films/{id}
Parmètre requis: id du film
* Éditer les informations d'un film spécifique
PUT api/films/{id}
https://four1f-tp1-claraquintela-1.onrender.com/api/films/{id}
Pametres requis: un objet film dans le corps de la rêquete:
            let film = {
                    titre: "titre",
                    description: "description",
                    annee: 2024,
                    realisation: "nom du réalisateur",
                    genre: "genre du film",
                    titreVignette: "titre de la vignette"
                  }
* Deleter un film spécifique
DELETE api/films/{id}
https://four1f-tp1-claraquintela-1.onrender.com/api/films/{id}
Parmètre requis: id du film

Utilisateurs

Pour avoir la liste de tous les utilisateurs
GET /api/utilisateurs
https://four1f-tp1-claraquintela-1.onrender.com/api/utilisateurs
Pour faire s'inscrire dans le site
POST /api/utilisateurs/inscription
https://four1f-tp1-claraquintela-1.onrender.com/api/utilisateurs/inscription
Pour faire le login
POST /api/utilisateurs/connexion
https://four1f-tp1-claraquintela-1.onrender.com/api/utilisateurs/connexion
